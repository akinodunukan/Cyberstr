# CyberStrongShield

CyberStrongShield is a cyber risk intelligence platform that helps organisations understand, prioritise, and reduce their cyber risk.

## Core Capabilities

- Cyber Health Scoring
- Risk Prioritisation
- Control Effectiveness Tracking
- Risk Simulation
- AI-powered Risk Explanation

## Architecture

- Backend: Spring Boot (Java)
- AI Service: FastAPI (Python)
- Frontend: React + TypeScript
- Database: PostgreSQL (RLS enabled)
- Storage: S3-compatible
- Auth: OAuth2 / OIDC
